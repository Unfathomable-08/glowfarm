# GlowFarm ✨

**Soft whispers of light chasing away the dark.**

GlowFarm is a modern, high-end lighting showcase e-commerce website that brings ethereal lighting products to life with breathtaking animations and immersive interactions.

Designed to evoke warmth, calm, and presence, the site features a poetic brand experience where every hover, theme switch, and light effect feels truly alive.

### ✨ Features
- Jaw-dropping animations & micro-interactions
- Dynamic lighting effects and hover states that feel magical
- Smooth dark/light theme switching
- Fully responsive design with buttery-smooth performance
- Elegant product showcase (Lumina Desk, Zenith Chandelier, Aura Floor, Nebula Pendant, and more)
- Poetic storytelling and premium testimonials

### 🛠️ Built With
- **Next.js** (App Router)
- **Framer Motion** — for advanced animations and gestures
- **Tailwind CSS** — for beautiful, responsive styling
- TypeScript

### 🎨 Design Philosophy
Before homes were flooded with harsh light and glowing screens, lamps created **warmth, calm, and presence**.  
GlowFarm brings that soul back — lighting that doesn’t just illuminate, but **slows time**.
